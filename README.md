### google_calendar_producer
將學校行事曆轉換成Google日曆格式
#### 使用方法
1. 開啟行事曆PDF檔全選複製
2. 在`input.txt`貼上
3. 執行`index.php`
4. 產生`output.csv`可匯入Google日曆
